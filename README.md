# scriba
Questo è il mio strumento principe per fare backup delle mie macchine. Puoi creare dei preset per i tuoi backups ed eseguirli quando vuoi o pianificarli con le utilità di pianificazione del tuo sistema operativo. Usa il potente strumento robocopy ed effettua il mirroring di strutture di cartelle fra 2 diversi supporti di memorizzazione.
